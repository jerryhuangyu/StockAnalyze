# Stock Insight App

## Technology

### stack

* MySQL
* Expressjs
* React
* Nodejs

### frontend

* RTK Query: state management for api call
* Recharts: chart
* Tailwindcss: css framework
* React router dom: navigation and routing(client side rendering)
* Driver: onboard guide

### backend

* Nodejs: runtime env
* Expressjs: backend framework
* MySQL: manage mysql database
